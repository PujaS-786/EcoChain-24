[1:00 - ARCHITECTURE]
(show assets/architecture_diagram.png)
Let us walk through the system architecture. The entire workflow is conducted by our Orchestrator Agent which coordinates our specialized specialist agents. First, incoming data goes through our Security Checkpoint where we check authorization and scrub data. Next, the Ingestion Agent normalizes invoices, which are calculated by the Calculation Agent. The calculation agent accesses custom emission datasets from our stdio MCP Server. From there, the Supplier Audit Agent aggregates sustainability tier profiles, while the Anomaly Detection Agent runs statistical baseline checks. If anomalies are found, the record is quarantined for human auditor verification. Finally, the Compliance Agent maps results to CSRD and GHG Protocol frameworks, and the Report Agent generates the board summaries.
